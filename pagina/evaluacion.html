La Importancia de las Matemáticas

Las matemáticas son fundamentales para el desarrollo intelectual de las personas, ayuda a ser lógicos, a razonar ordenadamente y a tener una mente preparada para el pensamiento, la crítica y la abstracción para la solucion de problemas.

Pensamiento Matemático

Es la sistematización y la contextualización del conocimiento de las matemáticas. Este tipo de pensamiento se desarrolla a partir de conocer el origen y la evolución de los conceptos y las herramientas que pertenecen al ámbito matemático.

Procedimientos Matematicos

  1. Herramientas que satisfacen una necesidad concreta de forma eficaz.
  2. Aplicacion o modificacion para la adecuacion a una nueva situacion.
  3. Modelos de resolucion de problemas.
  4. Verificacion de procesos correctos o incorrectos.
